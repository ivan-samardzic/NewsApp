### Newsletter App

This App is using third party API by which the data (the news) are fetched. 

App will use:
1. React (React Router, React Hooks, React JSX, React Link, React Props, React Stats and more)
2. Context Api 
3. CSS
4. GoogleFonts, FontAwesome and Bootstrap
5. Git and Postman
